SMARTWASTE MANAGEMENT USING IOT

A system is introduced to manage waste in large cities effectively monitoring 24Ã—7 manually. Here the problem of unorganized

and non-systematic waste collection is solved by designing an embedded IOT system that will monitor each dumpster individually

for the amount of waste deposited. A mechanical setup can be used for the wet and dry waste into separate containers. By using

sensors we can separate dry waste and wet waste. IR sensor is used to detect the presence of wet or dry waste. For detecting wet waste a

moisture sensor can be used. In this process, if only IR sensor is detected, motor will rotate in the direction of the dry waste container.

If both IR and moisture sensor detects the waste then it will go to the wet waste container. Both these containers are embedded with

ultrasonic sensors at the top, which are used for measuring distance. This makes possible to measure the amount of waste in the containers.

If one of the containers is full then an alert message will be sent to the corresponding person.
